A propos de Matrimap

Cet application a été développé dans le cadre de ma formation "Développeur Intégrateur Web - Full Stack Javascript" à l'IFOCOP de Paris, comme Projet Final pour présentation devant un jury. Son objectif est d’implémenter toutes les connaissances acquises pendant la formation et période d'aplication pratique en entreprise.

Matrimap est une « Single Page Application » développé utilisant Angular 8 comme framework front-end et NodeJS comme une API RESTful. L'application et l'api utilisent Socket.io aussi pour le chat.

Son visuel est basé sur la librairie de composants Angular Material.
